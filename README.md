# 🎉 Birthday Greeting Webpage

An interactive, customizable birthday greeting webpage with animations, music, and a modern glass UI design.

## ✨ Features
- 🎂 Personalized birthday message
- 🎵 Background music support
- 🎆 Confetti effects
- 🖼️ Image gallery with carousel & modal view
- 📱 Fully responsive (mobile-first design)
- 🎨 Modern glassmorphism UI

## 🚀 Demo
Open `index.html` in your browser.

## 📁 Project Structure
- `index.html` – Main webpage
- `guest.css` – Styling and UI effects
- `guest.js` – Interactions and animations
- `photo/` – Image assets

## 🛠️ Use Case
This project helps users create simple, shareable birthday greeting pages. It is also useful for beginners learning HTML, CSS, and JavaScript.

## 📌 Future Improvements
- Template customization
- User input for dynamic names/messages
- Hosting support (GitHub Pages)
- More animation effects
